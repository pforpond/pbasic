QB64 is a modern extended BASIC programming language that retains QBasic/QuickBASIC 4.5 compatibility and compiles native binaries for Windows, Linux, and macOS.

The _RED32 function returns the red component intensity of a 32-bit image or surface color.

Syntax

red32color& = _RED32(rgbaColor&)

Description

  • rgbaColor& is the 32-bit RGBA color value to retrieve the red component intensity value from.
  • RGBA color values are returned by the _PALETTECOLOR (function), POINT, _RGB, _RGB32, _RGBA or _RGBA32 functions.
  • LONG intensity values returned range from 0 (no intensity, not present) to 255 (full intensity).
